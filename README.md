## 说明
本项目来源于https://github.com/oucailab/OUC-LaTex-master
并在此基础上，严格按照中国海洋大学提供的Word版学位论文格式要求和外观重新进行了优化修改

## 主要改动如下：

1.新增**四级标题格式：\subsubsection{}**
2.新增**英文目录中摘要项：**
             Abstract (In Chinese)
             Abstract (In English)
3.新增**注释表**
             注释表
             Abbreviation
4.优化了**首页（cover.tex）的整齐度**
5.优化了**一级标题和页眉的距离**
6.优化了**各级标题的数字字体，改为了Times New Roman**
7.优化了**一级标题的字体间距**
...

## 如何使用
推荐使用 **Overleaf** 使用本模板，具体方法如下：

1. 下载项目，形成.zip 文件
2. 在 Overleaf 中上传该 .zip 文件以创建一个新 Overleaf 项目
3. 在 Overleaf 界面选择 "setting"
   - 选择 "Compiler" 为 "XeLaTeX"
   - 选择 "TeX Live version" 为 "2019" 或者更新的版本
5. 随后，可以正常编译成pdf
